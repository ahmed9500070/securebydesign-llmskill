## 📋 What is securebydesign-llmskill?

SecureByDesign-llmskill is a text file you give to an AI assistant, like ChatGPT or Claude. This file tells the AI how to check the security of the code it writes or recommends. It runs 25 security checks on every line of code the AI suggests.

The goal is to help keep your code safe without you needing to be an expert. You add this skill to your AI, and the AI uses it to catch common security mistakes automatically.

It works with any AI assistant that accepts skills in text file format. This makes it flexible and easy to use.

---

## ⚙️ How does it work?

The skill contains 25 security checks grouped into five layers. Each check points out a common security problem and guides the AI on how to fix it.

For example, one layer may check user input to stop dangerous data from getting into your program. Another layer may ensure passwords are handled safely.

The skill follows recognized security standards to make sure your code meets real-world rules.
